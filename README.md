# AidFlow Verify

Full-stack verified aid request platform.

## MVP Goal

Requesters submit essential aid requests, AI structures the request, admins review it, volunteers verify it, donors claim verified requests, and fulfillment is tracked with proof and audit logs.

## Tech Stack

- Frontend: Next.js, TypeScript, Tailwind CSS
- Backend: Python, FastAPI
- Database: PostgreSQL + PostGIS
- Auth: JWT, bcrypt
- AI: OpenAI API or Gemini API
- Testing: pytest, httpx
- Docker: Docker Compose
- CI/CD: GitHub Actions
